# Ride_Booking_Dashboard

📊 Ride Booking Analysis Dashboard (Power BI)

📝 Project Overview
This project is an interactive Ride Booking Dashboard developed using Power BI, designed to analyze ride booking operations across multiple dimensions such as vehicle type, revenue, cancellations, and customer satisfaction.
The dashboard provides a comprehensive view of business performance and helps in identifying trends, patterns, and key operational insights.


🚀 Key Features

🚗 Vehicle Type Analysis
Comparison of booking value across different vehicle types (Auto, Bike, eBike, Go Mini, Sedan, Uber XL)
Success booking vs total booking analysis
Average distance and total distance travelled insights

💰 Revenue Insights
Revenue trends over time (daily & monthly)
Revenue breakdown by payment method
Revenue per kilometer analysis
Peak booking periods identification

❌ Cancellation Analysis
Customer and driver cancellation reasons
Cancellation distribution using pie charts
Overall cancellation percentage tracking (~25%)

⭐ Ratings Analysis
Customer vs Driver ratings comparison
Vehicle-wise rating insights

📊 Summary Dashboard
Service Quality Score (Gauge Visual)
Customer Retention Rate (~30%)
Total Revenue per KM
Premium Peak Hour Index (~72%)
Booking trends by day parts (Morning, Afternoon, Evening, Night)

🛠️ Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Excel / CSV Dataset

📊 Key Insights
Auto and Go Mini contribute highest booking value
Digital payment methods dominate transactions
Evening time shows highest booking activity
Significant cancellations due to customer & driver-related issues
Strong correlation between service quality and customer retention
